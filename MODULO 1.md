# consideraciones ambientales

1. pruebas de infraestructura de red
	1. se evalua la postura de seguridad de la infraestructura de red y como puede defenderse de los ataques.
	2. dispositivos que se incluyen
		1. enrutador
		2. cortafuegos
		3. revursos de soporte
		4. servidores AAA
		5. IPS
2. pruebas basadas en aplicaciones
	1. se centra en probar las debilidades de seguridad en las aplicaciones empresariales.
		1. Debilidades a incluir
			1. Configuraciones incorrectas
			2. Problemas de validación de entrada
			3. Problemas de inyección.
			4. Fallas lógicas.
3. pruebas de penetración en la nube.
------
Existen métodos de prueba de penetración como los siguientes:
1. Prueba de entorno desconocido.
	1. Al evaluador se le proporciona una cantidad muy limitada de información ejemplo de ello podría ser:
		1. Direcciones IP
		2. Nombres de dominio.
2. Prueba de entorno conocido.
	1. Al evaluador se le entrega diagramas de red, direcciones IP, configuraciones y un conjunto de credenciales de usuario, opcional pueden entregarse código fuente de una aplicación de destino en caso de evaluar aplicaciones
3. Prueba de entorno parcialmente conocido.
	1. es posible que que se le entreguen credenciales de usuario pero no diagramas , es un enfoque híbrido entre conocido y desconocido.

### Estándares y metodologías.

1. MITRE ATT&CK
	1. Enumera las tácticas y técnicas que los adversarios utilizan mientras se prepara para un ataque incluida la recopilación de información.
2. OWASP
	1. Guía completa centrada en las pruebas de aplicaciones Web.
3. NIST SP 800-115
	1. Documento que proporciona a las organizaciones pautas sobre la planificación y realización de pruebas de seguridad de la información.
4. OSSTM
	1. Manual de metodología de pruebas de seguridad de código abierto.
5. PTES
	1. Proporciona información sobre tipos de ataques y métodos, y proporciona información sobre las últimas herramientas disponibles para lograr métodos de prueba descritos.
6. ISSAF
	1. Es otra metodología similar a las demás solo que con algunas fases adicionales.
